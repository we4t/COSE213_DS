후위 표기법은 괄호 없이 중위 표기법 식을 표기함.
우선순위가 없음. 무조건 왼쪽에서 오른쪽으로 읽어서 계산함.

중위 -> 후위 변환법
ex) a / b - c + d * e - a * c
1. 모든 계산을 괄호로 묶는다.
	((((a / b) - c) + (d * e)) - (a * c))
	// 항상 앞에서부터 묶어나간다.
	// de - ac를 먼저 묶지 않는다.

	((((a / b) - c) + (d * e)) - (a * c))
