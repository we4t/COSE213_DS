이중 결합 그래프 : biconnected graph
단절점이 없는 연결 그래프.
어떤 vertex나 edge를 지워도 항상 연결이 되는 그래프.
통신 네트워크에서 주로 사용. (하나 꺼져도 항상 통신 되도록)

이중 결합 요소 : biconnected components
무방향 연결 그래프에서 이중 결합 요소는 
biconnected maximal subgraph.

특징
1. 한 그래프의 두 이중 결합 요소는 
둘 이상의 정점을 공통으로 가질 수 없음.
2. 간선은 서로 다른 이중 결합 요소에 동시에 포함될 수 없음.

구하는 법 : g의 dfs spanning tree로 구함.
pre knowledge : 
그래프에서 A의 조상 : A 이전에 방문한 노드
그래프에서 A의 후손 : A 이후에 방문한 노드
< u가 v의 조상 : dfn(u) < dfn(v) >

1. v 정해서 dfs(v) 호출.
방문한 순서대로 dfn 붙임.
ex) dfn(v) = 0, dfn(v1) = 1, ...
이 때, 백엣지는 dfs 할 때
이전에 방문한 vertex를 이용하지 않은 edge로 방문할 수 있을 때
이용하지 않은 edge를 백엣지라고 함.

2. 모든 버텍스 마다 이 작업 수행.
low(u) = min{ 
dfn(u), 
min{low(w)| w는 u의 자식}, 
min{dfn(w)| (u, w)는 백엣지}
}
즉, low는
자기 자신의 dfn 이거나
자식 중에 가장 작은 값의 low 값이거나
백간선으로 도달할 수 있는 가장 작은 dfn 값

깔끔하게 수행하기 위해
leaf 노드부터 수행하는 것을 추천..

3. 만약
1) 
dfn(u) > low(w) : 자식을 나중에 방문해도 자식값이 작음.
자식(w)에서 조상으로 가는 다른 경로가 있음.
2) 
dfn(u) <= low(w) : 자식(w)에서 조상으로 가는 
다른 경로가 없음 (단절점 !)

정리 : 
단절점인 경우 : 
1) 두 개 이상의 자식을 갖는 스패닝 트리의 루트일 경우
2) 루트가 아니고 dfn(u) <= min(low(w)) 인
자식 w를 가지는 경우
